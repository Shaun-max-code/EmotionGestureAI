# AI-Based Emotion Recognition System

## Overview

This project uses Deep Learning and Computer Vision to detect facial emotions from images.

## Features

* Face Detection using OpenCV
* Emotion Recognition using TensorFlow
* FER2013 Dataset
* Streamlit Web Interface
* Confidence Score Visualization

## Technologies Used

* Python
* TensorFlow
* OpenCV
* Streamlit
* NumPy

## Dataset

FER2013 Facial Expression Dataset

## Future Enhancements

* Video Emotion Recognition
* Gesture Recognition
* AI Interview Performance Analyzer
